# User Management System (React + Node.js + MongoDB)

This is a basic full-stack CRUD application built using React.js, Node.js, and MongoDB.

## Features
- Add users
- View users
- Delete users

## Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB

## Description
This project demonstrates a simple user management system where users can be added, viewed, and deleted. The frontend is built using React.js, while the backend is developed using Node.js and Express.js. MongoDB is used for storing user data.

## How to Run

1. Clone the repository  
2. Install dependencies using `npm install`  
3. Start backend using `node server.js`  
4. Start frontend using `npm start`

## Project Structure
- backend/server.js
- frontend/App.js
